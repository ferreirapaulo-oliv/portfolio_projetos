Projetos de ciência de dados

Desenvolvimento de um Painel Executivo para Marketplace de restaurantes com o Streamlit

Nesse projeto, os conceitos de Programação em Python, Manipulação de dados, pensamento estratégico e lógica de negócio, junto com as ferramentas de desenvolvimento web como Streamlit e Github, foram usados para desenvolver um Painel de Controle Executivo com algumas métricas de uma empresa de marketplace de restaurantes.

O resultado desse final desse projeto foi um painel hospedado em um ambiente Cloud e disponibilizado através do Streamlit.

Lista de tecnologias utilizadas:

  • Python
  • Jupyter Lab
  • Terminal
  • Streamlit
  • Streamlit Cloud
